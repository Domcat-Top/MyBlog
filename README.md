# MyBlogs
## 描述
  ###### 一个简单的个人博客
## 实现方式

  ###### 基于Springboot实现。
  ###### 前后端交互使用Thymeleaf。
  ###### 数据库使用Mysql5.7，数据库工具使用Mybatis。
  ###### 首页实现方式采用Redis缓存。
  ###### 后台写入博客的方式集成了MarkDown插件，可以使用Makrdown编辑博客。
  ###### 整合了谷歌二维码生成工具，可以生成二维码并且上传到服务器，调取到前端页面展示。
